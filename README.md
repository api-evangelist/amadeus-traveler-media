# Amadeus Traveler Media API (amadeus-traveler-media)
The Amadeus Traveler Media API provides access to travel-related media content, including photos and ratings for points of interest and hotels worldwide.

**URL:** [Visit APIs.json URL](https://developers.amadeus.com)

## APIs

### Amadeus Safe Place API
Returns the safety rating of a location based on the geocode (latitude and longitude).

**Human URL:** [https://developers.amadeus.com/self-service/category/destination-content/api-doc/safe-place](https://developers.amadeus.com/self-service/category/destination-content/api-doc/safe-place)


#### Tags:

 - Travel, Safety, Location, Ratings

### Amadeus Points of Interest API
Find relevant points of interest (hotels, restaurants, attractions) within a specific area or radius.

**Human URL:** [https://developers.amadeus.com/self-service/category/destination-content/api-doc/points-of-interest](https://developers.amadeus.com/self-service/category/destination-content/api-doc/points-of-interest)


#### Tags:

 - Travel, Points of Interest, Location, Attractions

#### Properties

- [Documentation](https://developers.amadeus.com/self-service/category/destination-content/api-doc/points-of-interest)
- [OpenAPI](https://developers.amadeus.com/self-service/category/destination-content/api-doc/points-of-interest/api-reference)

### Amadeus Tours and Activities API
Find the best tours, activities, and attractions for travelers at a destination.

**Human URL:** [https://developers.amadeus.com/self-service/category/destination-content/api-doc/tours-and-activities](https://developers.amadeus.com/self-service/category/destination-content/api-doc/tours-and-activities)
